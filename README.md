# Funcionalidades do programa
```
* Carregar ficheiros de texto e objeto com os planteis de uma equipa e respetivos jogos
* Efetuar um jogo de futebol
   - Definir tática
   - Definir onze inicial
   - Fazer substituições
   - Criar jogadores
   - Criar equipas
```
